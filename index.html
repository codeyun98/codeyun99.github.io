MySQL基础

发表于 2021-12-23 分类于 SQL

MySQL基础

表中字段的增、改、删除

1. 新增字段

alter table 表名 add 字段名 字段类型;

2. 修改字段

2.1. 字段名称修改
alter table 表名 change 旧字段名 新字段名 字段类型;
2.2. 字段类型修改
alter table 表名 modify 字段名 新字段类型;

3. 删除字段

alter table 表名 drop column 字段名;
